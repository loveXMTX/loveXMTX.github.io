Task 1

Requirements
  • 150 words
  • 20min
  • Chart
  • Be objective，no subjective
Structure
  • Introduction
    • must be very clear
    • rewrite the prompt and/or add a bit of essential info
    • do not copy prompt
  • Overview
    • most important
    • Summary, no conclusion
    • signposting phrase “Overall”/“On the whole”
    • include comparisons
    • capture 2-3 key features
    • no including specific data or number
  • Body1
  • Body2
    • 2 paragraphs
    • include comparisons
    • include specific data or number
Timetables

  • 审题 1min

    注意年份，时态

  • 写 18min

  • 检查 1min

Preparation

  • 掌握结构
